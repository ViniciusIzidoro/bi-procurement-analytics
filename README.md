## Objetivo
Desenvolver uma solução de Business Intelligence para controle dinâmico de compras e análise de custos, permitindo a identificação rápida de gastos por fornecedor e a evolução temporal das aquisições.

## Funcionalidades Principais

| Recurso | Descrição |
| :--- | :--- |
| **KPI de Compra Total** | Visão consolidada do volume financeiro total de compras. |
| **Distribuição de Gastos** | Ranking de gastos por fornecedor para análise de concentração de custos. |
| **Evolução Temporal** | Gráfico de linha para acompanhamento mensal da performance de compras. |
| **Filtros Interativos** | Segmentação por Produto e Status da Entrega (Entregue/Cancelado). |

## Tecnologias Utilizadas
* **Power BI:** Ferramenta utilizada para a modelagem visual, criação de gráficos e filtros interativos.
* **Excel:** Base de dados estruturada utilizada como fonte primária para a alimentação e processamento das informações.
